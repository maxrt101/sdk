# SDK
SDK For embedded projects.  

### Short Description
The SDK is a collection of various modules, that helps creating embedded (or not) projects.  
It contains:
 - Libraries (log, shell, queue, ...)
 - Hal (uart, spi, i2c, ...)
 - Drivers (like for RA-02, AHT10, etc)
 - Platforms (STM32, ...).  
 - Build system (based on CMake).
