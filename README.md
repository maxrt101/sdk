# SDK
SDK For embedded projects.  

### Short Description
The SDK is a collection of various modules/drivers/etc, for embedded projects.  
It contains:
 - Libraries (log, shell, queue, ...)
 - Hal (uart, spi, i2c, ...)
 - Drivers (like for SX1278 (LoRa mode), AHT10, etc)
 - Platforms (STM32, ...).  
 - Build system (based on CMake).
